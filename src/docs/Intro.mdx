# card-ds

`card-ds` is a small design-system package built for a reusable product Card component.

The repo demonstrates the design-system workflow in a compact form:

- Design tokens are stored in `tokens/theme.json`.
- Style Dictionary generates CSS custom properties in `src/styles/tokens.css`.
- Components consume tokens through standard CSS.
- Storybook documents component usage and behavior.
- A local consuming app imports the built package through npm workspaces.

## Current Scope

This design system currently exports:

- `Card`
- `ProductList`
- `CardProps`
- `ProductListProps`

The package also exports `card-ds/styles.css`, which includes token custom properties and component styles.

## Intended Use

Use this package when a product experience needs consistent product-card presentation using the provided design tokens, focus treatment, spacing, shadows, and responsive list behavior.

This is intentionally small. The public API is kept narrow so the component is easy to adopt and review.
